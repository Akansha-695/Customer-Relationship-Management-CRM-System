## Project Overview
This project is a **Customer Relationship Management (CRM) System** designed using **MySQL** to manage customer-related data for a business.  
It demonstrates the use of **relational database concepts** such as table creation, constraints, relationships, and SQL queries.

The database stores information about **customers, leads, contacts, and sales opportunities** in a structured and organized manner.

---

## Objectives
- To design a relational database for customer relationship management  
- To practice core SQL concepts using MySQL  
- To implement relationships using primary and foreign keys  
- To track leads, customers, and sales opportunities using SQL queries  

---

## Technologies Used
- **Database:** MySQL  
- **Tool:** MySQL Workbench  
- **Language:** SQL

## Relationships
- One customer can have multiple contacts  
- One customer can have multiple opportunities  
- Leads can be converted into customers  
- Opportunities are linked to customers  

Foreign key constraints are used to maintain data integrity.
